  *  FORMULARIOS


 --- para criar um formulario começamos com a tag <form></form>

 --- <input type="" nome""> tag componente para formulario

    escrevendo text entre as aspas do type aparece uma caixa de texto a abaixo

  --- ex :  <input type="text" nome"">


 ---  para senha e recomendado usar o componente type="password"

  -- ex : <input type="password" nome""> -> assim a senha na fica visivel 

 --- para criar um button basta usar o componente button entre as aspas do type

 -- ex : <input type="button" nome"">

-- para colocar o atributo no button basta colocar a tag VALUE=""

 EX: <input type="button" value="logar">


 --- componente (RADIO)- aparece um circulo para marcar algo. 

  -- ex : <input type="radio" name=""> masculino <br>

 --  OBS: colocar sempre o mesmo name ex: <input type="radio" name="sexo"> masculino <br>

 --- para saber o que o usuatio escolher tem que colocar a tag (value="")

---  ex: <input type="radio" name="sexo" value="m"> masculino <br>  -> M para masculino
--- <input type="radio" name="sexo" value="F"> masculino <br> F para feminino
